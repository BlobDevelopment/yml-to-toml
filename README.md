# yml-to-toml

Convert YAML files to TOML files with ease!

## Install

### Global

```sh
$ npm install -g yml-to-toml
```

### Project

```sh
$ npm install --save-dev yml-to-toml
```

## Usage

### CLI

```sh
$ yml-to-toml <file>
```

### NPM script

```json
"scripts": {
	"publish": "yml-to-toml wrangler.yml && wrangler publish"
}
```
